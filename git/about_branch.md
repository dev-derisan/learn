# About Branch

- List all branches

```
git branch
```

- Craete new branch

```
git branch <branch-name>
```

- Switch to other branch

```
1. git checkout <branch-name>
2. git switch <branch-name>
3. git switch -c <branch-name> // Create and switch
```

- Delete branch

```
git branch -d <branch-name>
```

- Rename branch

```
git branch -m <new-name>
```
